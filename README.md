# PATAKY POOL #

Pataky pool is a website used to establish pool rankings.

## POINT SYSTEM ##
-Win: **5 points**

-Puting your ball: **1 points**

-Puting enemy ball: **-1 points**

-Puting the white ball: **-1 points**

-Puting the black ball (leading to a loss): **-5 points**
